# 用户 CRUD — 功能规格

## 概述
管理端用户管理功能，包括用户列表、搜索筛选、详情查看、角色配置和密码重置。

## 所属信息
- 模块：user-management
- 前置依赖：auth（Admin 角色）

## 用户故事
- 作为管理员，我希望查看和搜索所有系统用户
- 作为管理员，我希望管理用户角色和重置密码

## 页面内容

### 用户列表页
- 表格列：姓名、用户名、地区、角色、注册时间、操作
- 搜索框（按姓名/用户名搜索）
- 地区筛选下拉
- 分页控件

### 用户详情页/抽屉
- 基本信息编辑（姓名、地区、性别）
- 角色切换（USER ↔ ADMIN）
- 重置密码（设置新密码）
- 该用户的活动参与记录列表

## 交互说明
- 搜索实时筛选（防抖 300ms）
- 角色切换 → 二次确认 → 更新 → Toast 提示
- 重置密码 → 输入新密码 → 确认 → Toast 提示
- 批量删除 → 勾选多个用户 → 二次确认 → 删除
