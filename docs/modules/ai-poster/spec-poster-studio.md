# 海报工作台 — 功能规格

## 概述
员工基于自己的活动参与记录，选择风格和提示词，使用通义万相 Wan 2.7 Image Pro 生成个性化 CSR 纪念海报。

## 所属信息
- 模块：ai-poster
- 前置依赖：participation（需要有参与记录）

## 用户故事
- 作为员工，我希望基于我参与的活动生成一张专属纪念海报

## 页面内容

### 输入区（上方）
1. **选择活动**（下拉）：从"我已参与的活动"中选择
2. **风格选择**（6种预设风格卡片）：
   - 简约现代 / 水彩手绘 / 3D 立体 / 卡通插画 / 国潮 / 写实摄影
   - 点击即选，高亮显示当前选中
3. **自定义提示词**（可选文本输入框）
4. **「生成海报」按钮**

### 生成中状态
- 按钮变为 loading 状态（disabled + spinner）
- 提示语："AI 正在为你创作专属海报，请稍等约 20 秒..."
- 前端每 3 秒轮询 `GET /posters/{taskId}/status`
- 遇到 401/403 自动停止轮询

### 结果展示
- 海报图片展示
- 自动刷新下方历史画廊

## 交互说明
- 未选择活动时「生成海报」按钮置灰
- 生成中禁止重复提交
- 生成失败 → 错误提示 + 可重试
- 轮询超时（如 2 分钟）→ 提示"生成超时，请稍后重试"

## 验收标准
- [x] 活动下拉正确展示用户已参与的活动
- [x] 6 种风格卡片正确展示，点击选中
- [x] 自定义提示词输入正常
- [x] 生成海报调用 API 成功
- [x] 轮询状态正确展示（loading → 完成/失败）
- [x] 生成完成后海报正确展示
- [ ] 历史画廊自动刷新（poster-gallery 功能范畴，当前 MVP 未实现）
- [x] 401/403 时停止轮询并提示
