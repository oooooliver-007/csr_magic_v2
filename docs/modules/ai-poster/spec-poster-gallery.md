# 海报画廊 — 功能规格

## 概述
展示用户历史生成的所有 AI 海报，支持 Lightbox 大图预览。位于海报工作台下方和个人中心"我的海报"Tab。

## 所属信息
- 模块：ai-poster
- 前置依赖：poster-studio（需要有已生成的海报）

## 用户故事
- 作为员工，我希望浏览我历史生成的所有海报
- 作为员工，我希望大图预览和下载海报

## 页面内容
- 网格卡片布局
- 每张卡片：海报缩略图 + 活动名 + 生成时间
- 点击卡片 → Lightbox 大图预览
- Lightbox 内：大图 + 关闭按钮 + 下载按钮

## 交互说明
- 点击卡片打开 Lightbox（fadeIn + scaleIn 动画）
- 再次点击或点击关闭按钮 → 关闭 Lightbox
- 下载按钮 → 浏览器下载海报图片
- 无海报时显示空状态 + "去生成海报"引导

## 验收标准
- [ ] 画廊网格正确展示历史海报
- [ ] 卡片包含缩略图、活动名、生成时间
- [ ] Lightbox 大图预览正常打开/关闭
- [ ] Lightbox 有 fadeIn + scaleIn 动画
- [ ] 下载功能正常
- [ ] 空状态正确展示
- [ ] 移动端卡片自适应列数
