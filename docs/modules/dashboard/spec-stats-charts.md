# 统计看板 — 功能规格

## 概述
管理端数据看板，让管理员一眼了解 CSR 项目的整体运营状态，包括统计卡片、趋势图、分布图和排行榜。

## 所属信息
- 模块：dashboard
- 前置依赖：activity（活动数据）、participation（参与数据）

## 用户故事
- 作为管理员，我希望看到 CSR 项目的整体运营数据，以便制定决策

## 页面内容

### 顶部统计卡片（4个）
- 总活动数
- 累计参与人次
- 累计捐赠额
- 本月新增参与

### 参与趋势折线图
- 近 12 个月每月参与人次
- X 轴：月份，Y 轴：人次
- 支持 hover 查看具体数值

### 活动类型分布饼图
- 各模板类型活动占比
- 显示百分比和数量

### 最活跃员工 Top 10 列表
- 按参与次数排名
- 展示：排名、姓名、参与次数

### 近期活动状态列表
- 正在进行的活动及参与情况
- 展示：活动名、类型、时间、报名人数/上限

## 交互说明
- 页面加载时使用骨架屏（Skeleton）
- 图表支持 hover 查看详情
- 统计数据自动刷新（或提供手动刷新按钮）

## 验收标准
- [ ] 4 个统计卡片数据正确展示
- [ ] 参与趋势折线图正确渲染近 12 个月数据
- [ ] 活动类型分布饼图正确展示
- [ ] Top 10 员工列表正确排序
- [ ] 近期活动列表正确展示
- [ ] 加载时显示骨架屏
- [ ] 图表 hover 交互正常
- [ ] 移动端：统计卡片 2列→1列，图表堆叠
