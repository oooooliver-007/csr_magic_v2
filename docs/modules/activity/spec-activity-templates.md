# 5种活动模板 — 功能规格

## 概述
活动支持 5 种模板类型，每种模板决定员工报名时需要填写的表单字段和管理员创建活动时的配置项。

## 所属信息
- 模块：activity
- 前置依赖：activity-crud（模板在创建活动时选择）

## 用户故事
- 作为管理员，我希望选择合适的活动模板，以便收集不同类型活动所需的信息
- 作为员工，我希望看到简洁明了的报名表单，只填写当前活动类型所需的字段

## 模板定义

### 基础活动（BASIC）
- 员工填写：文字说明（可选）
- 适用场景：签名承诺、简单参与确认

### 捐赠活动（DONATION）
- 员工填写：捐赠金额（必填，数字）+ 留言（选填，文字）
- 适用场景：慈善捐赠、公益募资

### 志愿者活动（VOLUNTEER）
- 员工填写：服务时长（必填，数字，单位：小时）+ 照片上传（最多5张）
- 适用场景：社区服务、环保行动

### 签到活动（CHECKIN）
- 员工填写：自动记录签到时间 + 照片上传（可选）
- 适用场景：活动现场签到、打卡

### 自定义活动（CUSTOM）
- 管理员配置：字段名 + 字段类型（文字/数字/图片/开关）
- 员工填写：按管理员配置的字段动态生成表单
- 适用场景：特殊活动需求

## 交互说明
- 管理端创建活动时选择模板 → 预览该模板的字段说明
- 员工端活动详情页根据 templateType 动态渲染对应表单
- 自定义模板的 formSchema 存储在 activity.form_schema（JSONB）中

## 验收标准
- [x] 5种模板在管理端创建活动时可选择
- [x] 每种模板对应的员工端报名表单字段正确
- [x] BASIC 模板只有可选文字输入
- [x] DONATION 模板有金额（必填）和留言（选填）
- [x] VOLUNTEER 模板有时长（必填）和照片上传（最多5张）
- [x] CHECKIN 模板自动记录时间
- [x] CUSTOM 模板根据 formSchema 动态生成表单
- [x] 模板切换时表单正确更新
