# 活动详情 + 报名 — 功能规格

## 概述
员工端查看活动完整信息并完成报名。根据活动模板类型展示不同的参与表单。

## 所属信息
- 模块：activity
- 前置依赖：activity-list（从列表跳转）

## 用户故事
- 作为员工，我希望查看活动的完整信息，以便决定是否参与
- 作为员工，我希望通过表单快速完成活动报名

## 页面内容
- 活动封面大图
- 活动名称、类型标签、状态标签
- 所属事件（可点击跳转）
- 时间：开始时间 ～ 结束时间
- 活动描述（富文本）
- 参与表单（根据活动模板类型）：
  - **基础活动**：文字说明（可选填）
  - **捐赠活动**：捐赠金额（必填）+ 留言（选填）
  - **志愿者活动**：服务时长（必填）+ 照片上传（最多5张）
  - **签到活动**：自动记录时间，可选上传照片
  - **自定义活动**：按管理员配置的表单字段显示
- 底部操作栏：「立即报名」/「已报名 · 退出」/「活动已结束」
- 「AI 对话报名」按钮（与传统报名按钮并排）

## 交互说明
- 未报名 → 显示报名表单 + 报名按钮
- 有名额上限时显示「剩余名额：X」，满额则报名按钮置灰
- 已报名 → 显示已填信息（只读）+ 「退出活动」选项
- 审核驳回 → 显示驳回原因 + 「修改后重新提交」按钮
- 活动结束 → 只读模式
- 报名成功 → Toast 提示 + 更新状态

## 验收标准
- [ ] 活动详情所有字段正确展示
- [ ] 5种模板对应的参与表单正确渲染
- [ ] 报名成功后状态正确更新
- [ ] 退出活动功能正常
- [ ] 名额上限逻辑正确（满额置灰）
- [ ] 驳回状态正确显示原因
- [ ] AI 对话报名入口正确展示
- [ ] 移动端：单列布局，操作按钮固定底部
