# 活动 CRUD（管理端） — 功能规格

## 概述
管理端活动的完整增删改查功能。活动隶属于事件，支持 5 种模板类型选择。

## 所属信息
- 模块：activity
- 前置依赖：event（活动必须关联事件）

## 用户故事
- 作为管理员，我希望创建不同类型的 CSR 活动
- 作为管理员，我希望查看和编辑活动的详细信息
- 作为管理员，我希望管理活动的状态和可见性

## 页面内容

### 活动列表页
- 表格列：封面缩略图（40×40）、活动名称、所属事件、类型/模板、时间、状态、参与人数、操作
- 按事件筛选、按状态筛选
- 操作列：
  - 👁 View：只读详情抽屉（封面图、所有字段、参与人数进度条）
  - ✏️ Edit：预填表单抽屉，保存调 `PUT /activities/{id}`
- 分页控件

### 创建/编辑活动（侧边抽屉）
- 所属事件（下拉，必填）
- 活动名称（必填）
- 活动模板选择：基础 / 捐赠 / 志愿者 / 签到 / 自定义
- 开始/结束时间（日期选择器）
- 人数上限（可选，留空表示不限）
- 描述（富文本）
- 封面图上传（≤500KB，支持预览和清除）

### View 只读抽屉
- 封面图大图
- 所有字段展示
- 参与人数进度条（已报名/上限）
- 底部「去编辑」快捷跳转

## 交互说明
- 新建保存成功 → Toast + 刷新列表
- 编辑保存成功 → Toast + 刷新列表
- 删除 → 二次确认 → 删除 → Toast
- View 抽屉底部可快捷跳转 Edit 模式
