# 活动列表（员工端） — 功能规格

## 概述
员工端浏览全部可参与的 CSR 活动，支持搜索、筛选和分页。

## 所属信息
- 模块：activity
- 前置依赖：event（活动关联事件）

## 用户故事
- 作为员工，我希望浏览所有 CSR 活动，以便选择感兴趣的活动参与

## 页面内容
- 搜索栏（按活动名称搜索）
- 筛选条件：活动类型（5种模板）、状态（进行中/即将开始/已结束）、所属事件
- 活动卡片网格（2-3列）
- 每张卡片包含：封面图、活动名称、所属事件名、类型标签、时间、当前报名人数、「查看详情」按钮
- 分页控件
- 空状态：插画 + "暂无可参与的活动"

## 交互说明
- 搜索实时筛选（防抖 300ms）
- 筛选条件变化时重置到第一页
- 点击活动卡片或「查看详情」→ 跳转活动详情页
- 活动状态标签颜色区分：进行中（绿色）、即将开始（蓝色）、已结束（灰色）

## 验收标准
- [ ] 活动卡片网格正确渲染
- [ ] 搜索按名称筛选正常工作
- [ ] 类型、状态、事件筛选正常工作
- [ ] 分页正常工作
- [ ] 空状态正确展示
- [ ] 点击卡片正确跳转到活动详情页
- [ ] 移动端：卡片切换为单列布局，筛选可折叠
