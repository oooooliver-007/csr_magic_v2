# 个人设置 — 功能规格

## 概述
员工端个人信息管理，包括修改昵称、真名和密码。员工端和管理端均可访问。

## 所属信息
- 模块：user-profile
- 前置依赖：auth（认证）

## 用户故事
- 作为员工，我希望修改我的个人信息以保持准确
- 作为员工，我希望定期修改密码以保障账号安全

## 页面内容
- 个人信息卡（头像、姓名、用户名、地区）
- 编辑区域：
  - 昵称（可修改）
  - 真名（可修改）
  - 地区（下拉选择）
  - 性别（可选）
- 密码修改区域：
  - 当前密码（必填，验证）
  - 新密码（必填，≥6位）
  - 确认新密码（必填，与新密码一致）
- 「保存」按钮

## 交互说明
- 个人信息和密码修改分开提交
- 保存成功 → Toast 提示
- 密码修改时验证当前密码是否正确
- 新密码与确认密码不一致 → 行内提示

## 验收标准
- [ ] 个人信息正确展示当前数据
- [ ] 昵称和真名修改功能正常
- [ ] 密码修改需验证当前密码
- [ ] 新密码与确认密码一致性校验
- [ ] 保存成功有 Toast 提示
- [ ] 员工端和管理端均可访问
