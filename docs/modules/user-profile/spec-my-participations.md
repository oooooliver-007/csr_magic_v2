# 我的参与记录 — 功能规格

## 概述
员工端个人中心的参与记录 Tab，展示用户的所有活动参与历史、审核状态和贡献统计。

## 所属信息
- 模块：user-profile
- 前置依赖：participation（参与记录数据）

## 用户故事
- 作为员工，我希望查看我的所有活动参与历史和审核状态
- 作为员工，我希望了解我的累计贡献数据

## 页面内容

### 贡献统计（数字卡片）
- 参与活动数
- 累计志愿时长
- 累计捐赠总额

### 参与记录列表（Tab 1）
- 每条记录：活动名、参与时间、状态标签、活动类型图标
- 状态标签颜色：待审核（黄色）、已通过（绿色）、已驳回（橙色）
- 已驳回的记录显示「查看原因并重新提交」链接
- 点击记录 → 查看完整详情
- 右上角「导出记录」按钮（导出 CSV）

### 我的海报（Tab 2）
- 历史生成的所有海报（卡片网格）
- 每张卡片：海报缩略图、对应活动名、生成时间
- 「下载」「分享」按钮
- 右上角「去生成海报」快捷入口

## 交互说明
- Tab 切换不刷新页面，保持数据
- 导出记录 → 生成并下载 CSV 文件
- 点击海报缩略图 → Lightbox 大图预览
