# 业务术语表

> 统一项目中的业务术语，避免歧义。前端/后端/文档必须使用一致的术语。

## 核心业务概念

| 中文 | 英文 | 说明 |
|------|------|------|
| 事件 | Event | CSR 大事件，是多个活动的容器（如"2026春季CSR月"） |
| 活动 | Activity | 具体的 CSR 活动，隶属于某个事件（如"春季植树活动"） |
| 参与记录 | Participation / UserActivity | 员工参与某个活动的记录 |
| 报名 | Signup | 员工提交参与表单的动作 |
| 审核 | Review | 管理员审批参与记录的动作 |
| 模板 | Template | 活动表单模板，决定员工需填写的字段 |

## 活动模板类型

| 中文 | 英文（templateType） | 说明 |
|------|---------------------|------|
| 基础活动 | BASIC | 仅文字说明（可选填） |
| 捐赠活动 | DONATION | 捐赠金额（必填）+ 留言（选填） |
| 志愿者活动 | VOLUNTEER | 服务时长（必填）+ 照片上传（最多5张） |
| 签到活动 | CHECKIN | 自动记录时间，可选上传照片 |
| 自定义活动 | CUSTOM | 管理员配置的自定义表单字段 |

## 参与状态

| 中文 | 英文（state） | 说明 |
|------|-------------|------|
| 待审核 | PENDING | 员工已报名，等待管理员审核 |
| 已通过 | APPROVED | 管理员审核通过 |
| 已驳回 | REJECTED | 管理员驳回，需填写驳回原因 |
| 重新提交 | RE_SUBMITTED | 员工修改后重新提交（Phase 4） |

## 用户角色

| 中文 | 英文（role） | 说明 |
|------|-----------|------|
| 普通员工 | USER | 浏览和参与 CSR 活动 |
| 管理员 | ADMIN | 管理活动、审核参与、查看数据 |

## AI 功能

| 中文 | 英文 | 说明 |
|------|------|------|
| AI 对话报名 | AI Chat Registration | 通过自然语言对话完成报名（Qwen） |
| AI 海报工作台 | AI Poster Studio | 生成个性化活动纪念海报（Wan 2.7） |
| 海报画廊 | Poster Gallery | 历史生成海报的展示列表 |

## 通知类型

| 中文 | 英文（type） | 说明 |
|------|-----------|------|
| 报名成功 | SIGNUP_SUCCESS | 员工报名后自动发送 |
| 审核通过 | REVIEW_APPROVED | 管理员通过后发送 |
| 审核驳回 | REVIEW_REJECTED | 管理员驳回后发送 |
| 活动提醒 | ACTIVITY_REMINDER | 活动即将开始时发送 |
